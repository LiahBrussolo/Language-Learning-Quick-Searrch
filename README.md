A Chrome extension, made specifically for language learning.

### How it works:
Select a single word and it's immediately searched on pre-selected reference websites. Pre-select a maximum of 20 websites.

Example: If I select a word and have 2 websites -- for instance, a dictionary and a translation website -- selected, then 2 new tabs open. The word is entered into each site and searched so that when you visit the tab, the result is ready.

### Other features:
- Selected word(s) are copied to clipboard. When you select a sentence with a word in context, the entire example is instantly copied, ready to paste into Anki, study notes, etc.
- Download a list of the words searched during the day
- Extension can be opened in its own tab to more easily manage the websites
- It's easy to add and exclude websites, possible to make changes while on the website of interest

### For PDFs & non-scriptable pages
1. Right-click selected text and click Search with Multi-Search
2. Select text, then press Ctrl+Shift+F
